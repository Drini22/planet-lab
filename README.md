# planet-lab

Planet interiors, live — a single self-contained page:

- **Explore** — an interactive mass–radius explorer (real RK4 integration, Seager 2007 EOS fits)
- **Equations** — the three structure equations, derived step by step
- **Methods** — a numerical-methods playground: the exactly solvable planet, method
  families order by order, a live work–precision diagram, and the measured
  implementation ladder (Python → CUDA)
- **Planet lab** — design a material ρ(P) = ρ₀ + c·Pⁿ and watch its planets, plus a
  GPU-precomputed map of every material's biggest planet
- **Quiz** — self-testing

Everything runs in the browser; no dependencies, no server. Built as interactive
preparation for a PhD interview (University of Bern, July 2026).
